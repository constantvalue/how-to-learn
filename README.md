# Проектная работа "Научиться учиться"
## Анимации, шрифты, фреймы
___
Учебный проект, нацеленный на изучение основных технолгий верстки.
В ходе выполнения проектной работы были использованы следующие технологии и инструменты:
1. Язык разметки HTML
2. Язык стилей CSS
3. Примененена методология БЭМ. Построена Nester Структура проекта.
4. Использована Flexbox верстка.
5. Использован подход к верстке, подразумевающий использование семантических тегов.
6. Использованы трансформации вместе с Keyframes для анимации геометрических фигур.
7. С помощью iframe интегрировано Youtube видео.
___

В дальнешем планируется:
1. Добавить в проект формы с помощью HTML, CSS и JS.
2. Реализовать кроссбраузерность с помощью вендорных префиксов. Использовать для этого автопрефиксер.
